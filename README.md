# 🤖 SupplyChains Bot

Telegram-бот для ведения канала [@supplychains](https://t.me/supplychains) — про игры в логистику и цепи поставок.

## Возможности

- 🔍 **Поиск новостей** — автоматический сбор новостей по логистике, играм, образованию
- 📝 **Генерация постов** — готовые тексты по шаблонам (новости, обзоры, кейсы, викторины)
- 👨‍💼 **Согласование с админом** — посты приходят на проверку перед публикацией
- 📅 **Планировщик** — публикации по расписанию в оптимальное время
- 📊 **Статистика** — отслеживание публикаций и эффективности

## Быстрый старт

### 1. Создать бота

1. Напишите [@BotFather](https://t.me/BotFather)
2. Создайте нового бота: `/newbot`
3. Получите **токен** (вида `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### 2. Добавить бота в канал

1. Откройте настройки канала @supplychains
2. Добавьте бота администратором с правами:
   - ✅ Отправка сообщений
   - ✅ Редактирование сообщений
   - ✅ Удаление сообщений
   - ✅ Прикрепление файлов

### 3. Получить ID канала

- Если канал публичный (`@supplychains`) — используйте `@supplychains`
- Если приватный — узнайте числовой ID через [@userinfobot](https://t.me/userinfobot)

### 4. Установить зависимости

```bash
cd projects/supplychains-bot
pip install -r requirements.txt
```

### 5. Запустить

```bash
export TELEGRAM_BOT_TOKEN="123456789:ABC..."
export ADMIN_CHAT_ID="12345678"  # Ваш личный chat_id
export CHANNEL_ID="@supplychains"

python bot.py
```

## Управление ботом

Бот работает в двух режимах:

### Режим 1: Автоматический (по расписанию)

Бот сам ищет новости, формирует посты и присылает вам на согласование.

### Режим 2: Ручной (команды)

Отправьте боту в личку:

| Команда | Действие |
|---------|----------|
| `/start` | Приветствие и справка |
| `/force_fetch` | Найти новости сейчас |
| `/schedule` | Показать запланированные посты |
| `/stats` | Статистика канала |

### Согласование поста

Когда бот присылает черновик, у вас есть кнопки:

- ✅ **Публиковать** — сразу в канал
- ✏️ **Редактировать** — отправьте исправленный текст
- ❌ **Пропустить** — удалить черновик
- ⏰ **Отложить** — в запланированные

## Настройка контента

### Добавить свои источники

Отредактируйте `news_fetcher.py`:

```python
RSS_SOURCES = [
    "https://logist.ru/rss",
    "https://supplychaindigital.com/rss",
    # Добавьте свои
]
```

### Изменить расписание

В `content_manager.py`:

```python
CONTENT_CALENDAR = [
    {"day": "monday", "type": "news", "topic": "logistics_tech"},
    # Измените под свои нужды
]
```

### Изменить шаблоны постов

В `content_manager.py` — раздел `TEMPLATES`.

## Деплой на сервер

Для постоянной работы запустите через systemd или Docker:

```bash
# Systemd service
sudo cp supplychains-bot.service /etc/systemd/system/
sudo systemctl enable supplychains-bot
sudo systemctl start supplychains-bot
```
